This repository is a starter for a cartoon-kungfu themed site for the token `$BOBANAD`.

Quick start

1. Install dependencies:

```powershell
cd "c:\MemeCoin Web portofolio\BOBANAD"
npm install
```

2. Run development server:

```powershell
npm run dev
```

3. Build for production:

```powershell
npm run build
npm run preview
```

Notes

- Replace Dexscreener iframe `src` attributes in `src/components/Hero.jsx` and `src/components/ChartTokenomics.jsx` with your token-specific Dexscreener URL.
- Customize tokenomics and links in `ChartTokenomics.jsx` and `LoreSocial.jsx`.
- Deploy to Vercel/Netlify by connecting the repo and running `npm run build` as the build command.
